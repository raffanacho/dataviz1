2)Add meaningful colum:
-Insert a Country column
-Complete the column using the Location info
3)Create Pivot Table
-Rows: Country
-Values: Country, Summarize by: CountA
4)Google Sheets Chart:
-Create a Map View (bubble) using the pivot table
5)Embed the chart in this GitHub Page!
